# Library Management System

This project is a full-stack web application developed for managing a library system.
It allows administrators and staff to manage books, categories, members, loans, returns, fines, and reports.

## Features
- User authentication and authorization
- Book management
- Category management
- Author and publisher management
- Member management
- Borrow and return books
- Fine calculation for late returns
- Search and filtering
- Reports and dashboard

## Technologies
- ASP.NET Core Web API / MVC
- SQL Server
- Entity Framework Core
- JWT Authentication or Identity
- HTML, CSS, JavaScript / React

## Project Goal
The goal of this project is to digitalize the main processes of a library and make book and member management easier and more efficient.
